# Лабораторная работа

## Описание задания

Вам предоставлен набор данных, который содержит изображения лиц людей, их возраст и пол.

Необходимо придумать, описать кейс/задачу и решить её, для которой требуется обучить многометочный мультиклассовый классификатор изображений.

## Пример кейса

Например, вы работаете в организации, которая занимается выдачей пособий для различных категорий людей, таких как молодежь, люди среднего возраста, люди пожилого возраста, люди старческого возраста и старше.

Для каждой категории людей в организации имеются специалисты, которые находятся в разных частях здания. При этом для мужчин и женщин в каждой категории имеются отдельные специалисты. Кроме того, люди, пришедшие с детьми, обслуживаются отдельно.

В результате требуется написать систему, которая будет определять категорию человека по внешнему виду и рекомендовать соответствующего специалиста.

## Необходимо выполнить

- **[Опционально]** Придумать и описать кейс/задачу, для решения которой требуется мультиклассовый классификатор изображений.
- Разработать архитектуру нейронной сети, которая будет получать на вход изображение и выдавать результат многометочной мультиклассовой классификации этого изображения.
- Можно и нужно использовать предобученный классификатор изображений в качестве `backbone`-модели.
- В примере, описанном выше, модель должна определять:
  - возрастную категорию (`ребенок`, `молодежь`, `средний`, `пожилой`, `старческий`);
  - пол.
- Подобрать параметры и обучить несколько моделей с одной базовой архитектурой, но различными параметрами.
- Сравнить полученные модели и продемонстрировать результаты.
