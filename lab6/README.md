# Лабораторная работа 6

## Задание

Ваша задача — обучить модель классификации комментариев на позитивные и негативные.

Для этого предлагается использовать набор данных с разметкой о токсичности текста.

## Необходимо выполнить

- Предобработать данные.
- Обосновать выбор методов предобработки текста.
- Векторизовать текст.
- Подобрать параметры и обучить модель (можно использовать `BERT`).
- Убедиться, что значение `F1` не менее `0.75`.
